# QueDJ
## Full Stack CRUD App

### Setup Instructions
- Run NPM install inside root folder
- Create Postgress Database quedj
- Run db/Schema.sql
- Run Node or Nodemon App.js
- Enjoy :D

### Technologies used:
- HTML 5
- CSS 3
- jQuery
- Javascript
- Node.Js
- Express
- Bycrypt
- Node-Fetch
- Method-override
- Mustache-express
- PG-Promise

# MVP List
- Sign up/Login
- Storing songs in database
- Deleting songs from database
- Song Request using itunes api
- Updating Profile Details
